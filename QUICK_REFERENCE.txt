                    🔌 API ENDPOINTS (4 TOTAL)

1️⃣ LOGIN
   POST /api/client/session/login
   ├─ Request: {pc_name, member_id, duration_minutes}
   ├─ Response: {success, session, message}
   └─ Does: Create session, update PC status, log activity

2️⃣ LOGOUT
   POST /api/client/session/logout
   ├─ Request: {session_id, pc_name}
   ├─ Response: {success, message, total_cost}
   └─ Does: End session, calculate cost, update PC status

3️⃣ GET STATUS
   GET /api/client/session/{sessionId}
   ├─ Request: (GET param only)
   ├─ Response: {success, session}
   └─ Does: Return current session status

4️⃣ HEARTBEAT
   POST /api/client/session/heartbeat
   ├─ Request: {session_id}
   ├─ Response: {success, remaining_minutes}
   └─ Does: Update last_heartbeat, log activity
