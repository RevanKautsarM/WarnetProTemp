================================================================================
                    FIX: PAGE EXPIRED ERROR
================================================================================

ERROR: "419 Page Expired" atau "Token Mismatch"

PENYEBAB:
1. CSRF token tidak valid
2. Session expired
3. APP_KEY corrupt
4. Middleware config salah
5. Session storage error

SOLUSI:

================================================================================
                        SOLUSI #1: REGENERATE APP_KEY
================================================================================

APP_KEY adalah kunci enkripsi session dan CSRF token.
Jika corrupt, semua session invalid.

STEP 1: Buka command prompt

  cd C:\WarnetPro\WarnetPro

STEP 2: Generate APP_KEY baru

  php artisan key:generate

  Output:
  INFO  Application key set successfully.

STEP 3: Cek .env

  File: C:\WarnetPro\WarnetPro\.env
  
  Pastikan ada baris:
  APP_KEY=base64:xxxxxxxxxxxxxxxxxxxxxxxxxxxxx
  
  (Sebelumnya mungkin kosong atau invalid)

STEP 4: Clear cache

  php artisan cache:clear
  php artisan config:clear
  php artisan route:cache

STEP 5: Delete session files

  Folder: C:\WarnetPro\WarnetPro\storage\framework\sessions
  
  Hapus SEMUA file di folder tersebut!

STEP 6: Close browser sepenuhnya

  Close ALL tabs dengan localhost:8000
  Keluar dari browser

STEP 7: Buka browser baru dan login

  http://localhost:8000/login
  Email: admin@warnet.com
  Password: password

================================================================================
                        WHY PAGE EXPIRED HAPPENS
================================================================================

"Page Expired" terjadi ketika:

1. CSRF TOKEN MISMATCH
   - Terjadi saat APP_KEY berubah
   - Session lama tidak match dengan APP_KEY baru
   - Fix: Generate key + clear session

2. SESSION EXPIRED
   - Default timeout 120 menit
   - Atau session file hilang
   - Fix: Delete session files, restart

3. SESSION DRIVER WRONG
   - Configured tapi folder/file tidak ada
   - Fix: Create folder, set correct driver

4. CACHE ISSUE
   - Old config cache masih di memory
   - Fix: Clear config cache

5. MIDDLEWARE ERROR
   - VerifyCsrfToken middleware corrupt
   - Fix: Clear cache, restart
